# Vectors

### Angle between 2 vectors

**in 2D space **:

If vectors **a** = [xa, ya], **b** = [xb, yb], then:

```
α = arccos[(xa * xb + ya * yb) / (√(xa2 + ya2) * √(xb2 + yb2))]
```



**in 3D space :**

If vectors **a** = [xa, ya, za], **b** = [xb, yb, zb], then:

```
α = arccos[(xa * xb + ya * yb + za * zb) / (√(xa2 + ya2 + za2) * √(xb2 + yb2 + zb2))]
```



since `α = arccos[(a · b) / (|a| * |b|)]`
